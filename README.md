# 🌦 Weather App

A responsive weather web application that allows users to search any city and view:

- Current weather conditions
- "Feels like" temperature
- Humidity, pressure, wind speed, cloud %
- 5-day forecast
- Air pollution levels (AQI + pollutants)

---

## 🚀 Features

✔ Real-time weather data  
✔ 5-day forecast display  
✔ Air Quality Index with pollutant details  
✔ Unit selection (Celsius/Fahrenheit)  
✔ Clean UI with card-based layout  
✔ Responsive design  

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, EJS
- Backend: Node.js, Express.js
- API: OpenWeather API
- Templating: EJS
- Styling: Custom CSS

git clone https://github.com/Sumitsingh39499/weather-website
.git

Install dependencies
npm install

Create `.env` file and add your API key
